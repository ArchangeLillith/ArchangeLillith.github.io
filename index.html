About

I'm a fullstack web developer with many projects under my belt and knowledge of multiple front end frameworks. I enjoy coding websites to be very user friendly and a joy to visit.


I started my journey in 2022, learning HTML and CSS and transitioning to JavaScript in early 2023. After long talks with my now husband and family I decided to pursue web develoment as a career, knowing my prior medical career wouldn't translate well into my future in Canada.


In 2023 I attended MIT's Full Stack Coding program specifically for women and learned a lot! I earned my certificate from that, took a break for our wedding, then hit the ground running.


StarDash was my first large scale project, and I learned so much from it. Based in React, it made me contemplate the best way to store large amounts of data, keep data secure, handle user logins, state management, as well as the overall design of the website and project managemnet! I used a Trello board as a progress tracker and the designs were inspired by the game it was for, so any of the UI from Project Sekai was my inspiration.


After StarDash was launched, I've been focusing on other frameworks. My first real Angular project that wasn't a tutorial was my Tanto Cuore randomizer that I use for playing the card game with friends. I'm currently revisiting a couple projects that I chose to put a pause on while I focused on my MIT certificate, my slots machine project and Medieval Times.
